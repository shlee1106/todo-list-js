# todo-list-js


## 📌 프로젝트 개요
* **프로젝트 목적:** 나만의 웹페이지를 통해 할 일 목록을 직관적으로 정리하고 관리
* **프로젝트 간단한 소개:** DOM 조작과 LocalStorage를 활용한 자바스크립트 투두리스트 과제 제출
* **개발 기간:** 2026.05.12 ~ 2026.05.28
* **핵심 기술:**  DOM조작, LocalStorage활용, JSON 객체 직렬화, 이벤트 처리, 배열 메서드 및 제어문, 동적 스타일, 사용자 경험고려
