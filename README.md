# todo-list-js


## 📌 프로젝트 개요
* **프로젝트 목적:** 나만의 웹페이지를 통해 할 일 목록을 직관적으로 정리하고 관리
* **프로젝트 간단한 소개:** DOM 조작과 LocalStorage를 활용한 자바스크립트 투두리스트 과제 제출
* **개발 기간:** 2026.05.12 ~ 2026.05.28
## 🛠️ 핵심 기술 및 구현 기능 

### 1. Web Storage API를 활용한 데이터 지속성 
* **LocalStorage 활용:** 브라우저가 닫히더라도 데이터가 유지되도록 `localStorage`를 사용했습니다.
* **데이터 직렬화 및 역직렬화:** * 객체 배열 데이터를 저장할 때는 `JSON.stringify()`를 통해 문자열로 변환(직렬화)하여 저장합니다.
  * 데이터를 불러올 때는 JSON.parse()를 통해 다시 자바스크립트 객체 배열로 복원(역직렬화)합니다.
  * 단축 평가(`|| []`)를 활용하여 저장된 데이터가 없을 경우 빈 배열로 안전하게 초기화합니다.

### 2. 동적 DOM 조작 및 화면 렌더링 
* **컴포넌트 기반 데이터 출력:** 배열 데이터(`todos`)의 상태를 기반으로 템플릿 리터럴(Template Literal)을 사용해 HTML 태그를 동적으로 생성하고, innerHTML을 통해 화면에 렌더링합니다.
* **초기화 및 호이스팅(Hoisting) 활용:** 브라우저가 로드될 때 호이스팅 특성을 활용하여 최초 1회 `writeTodo()` 함수를 즉시 호출, 기존 데이터를 화면에 뿌려줍니다.

### 3. 배열 메서드 활용
* **조회 (Read):** forEach 메서드를 순회하며 데이터 배열 안의 객체(`item`)와 인덱스(`index`)에 접근해 리스트를 출력합니다.
* **추가 (Create):** push() 메서드로 새로운 할 일 객체(내용, 날짜, 완료 여부)를 배열에 추가합니다.
* **삭제 (Delete) 및 전체 초기화:** 특정 인덱스의 데이터를 `splice(index, 1)`로 제거하며, 전체 초기화 시에는 배열을 빈 배열(`[]`)로 비우고 `removeItem()`을 수행합니다.

### 4. 사용자 경험(UX) 및 이벤트 처리
* **키보드 이벤트 처리:** onkeydown 이벤트를 감지하여 사용자가 입력창에서 Enter 키를 눌렀을 때도 할 일이 추가되도록 구현했습니다.
* **예외 처리 및 유효성 검사 (Validation):** * 사용자 이름 입력 및 할 일/날짜 등록 시 trim() 메서드로 앞뒤 공백을 제거합니다.
  * 빈 값 입력이나 취소(null)가 들어왔을 때 alert() 경고 창이나 기본값('게스트') 처리를 통해 예외를 방지합니다.
* **토글 및 수정 (State Toggle & Update):** * 체크박스 선택 시 논리 부정 연산자(`!`)를 사용해 완료 상태(true/false)를 반전시키고, classList.toggle('done')으로 CSS 스타일을 동적으로 제어합니다.
  * prompt() 창을 활용해 기존 데이터를 미리 보여주고, 사용자가 수정한 내용을 실시간으로 동기화합니다.
